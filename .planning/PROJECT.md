# BeautyLanding

> Конструктор лендингов для косметологов и бьюти-мастеров

## 🎯 Цель проекта

Создать простой и доступный инструмент для косметологов, позволяющий им создавать профессиональные лендинги без технических знаний и больших затрат.

## 📊 Бизнес-контекст

### Проблема
Косметологи и бьюти-мастера нуждаются в онлайн-присутствии для привлечения клиентов, но:
- Найм веб-разработчика стоит дорого ($500-2000+)
- Существующие конструкторы (Tilda, Wix) слишком сложные
- Универсальные шаблоны не подходят для специфики бьюти-индустрии
- Нужны готовые решения "из коробки"

### Решение
Специализированный конструктор лендингов с:
- Готовыми шаблонами для бьюти-индустрии
- Максимально простым интерфейсом редактирования
- Встроенными формами записи и галереями работ
- Доступной ценой (one-time purchase)

### Целевая аудитория

**Основная (B2C):**
- Косметологи-частники
- Мастера инъекционной косметологии
- Массажисты
- Визажисты
- Мастера по уходу за лицом и телом

**Характеристики:**
- Возраст: 25-45 лет
- Преимущественно женщины
- Малый бизнес или самозанятые
- Ограниченный бюджет на маркетинг
- Минимальные технические навыки
- Активны в Instagram/WhatsApp

### Конкуренты

| Конкурент | Плюсы | Минусы | Наше преимущество |
|-----------|-------|--------|-------------------|
| Tilda | Красивые шаблоны | Сложный интерфейс, дорого ($15-25/мес) | Проще + дешевле |
| Wix | Много возможностей | Перегружен функциями, медленный | Специализация на бьюти |
| ThemeForest | Дешевые шаблоны | Нужны технические навыки для настройки | Готовое решение без кода |
| Instagram | Бесплатно | Не профессионально, нет своего сайта | Полноценный лендинг |

## 🛠 Технологический стек

### Frontend
- **Framework:** React 18
- **Styling:** Tailwind CSS
- **State Management:** Zustand (легче Redux для начинающих)
- **Routing:** React Router v6
- **Forms:** React Hook Form
- **Drag & Drop:** react-beautiful-dnd (для редактора)

### Backend
- **Runtime:** Node.js 20 LTS
- **Framework:** Express.js
- **Database:** PostgreSQL (Supabase free tier)
- **ORM:** Prisma
- **Authentication:** JWT + bcrypt
- **File Upload:** Multer + Cloudinary

### Infrastructure
- **Hosting Frontend:** Vercel (free tier)
- **Hosting Backend:** Railway или Render (free tier)
- **Database:** Supabase (free tier, 500MB)
- **File Storage:** Cloudinary (free tier, 25GB/мес)
- **Email:** SendGrid (free tier, 100 emails/день)

### DevOps
- **Version Control:** Git + GitHub
- **CI/CD:** GitHub Actions
- **Monitoring:** Sentry (free tier)
- **Analytics:** Google Analytics 4
